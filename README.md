# A股起涨点扫描系统 V3

基于大涨股起涨点特征的全市场扫描系统，支持 Streamlit Web Dashboard 可视化。

## 功能

- 🚀 **起涨点扫描**: 基于 121 只大涨股特征，一键扫描全市场起涨点候选
- 📊 **起涨点图形**: 交互式 K 线图 + MACD/KDJ/RSI 指标 + 模板对比
- 🏆 **大涨股模板**: 历史大涨股起涨点特征统计与可视化
- 📋 **股票池管理**: 自定义分组管理关注股票
- 📈 **技术分析**: 单只股票深度技术分析

## 安装

```bash
pip install -r requirements.txt
```

## 使用

```bash
# 初始化数据库
python init_system.py

# 启动 Dashboard
streamlit run app/dashboard.py --server.port=8501

# 或运行全量数据采集
python run_full_collection.py
```

## 数据来源

- AKShare（新浪接口）
- 覆盖 A 股 2000 年至今全部日线数据
- 包含 KDJ、MACD、RSI、布林带、均线、量能等技术指标

## 起涨点特征（基于大涨股统计）

| 特征 | 占比 |
|------|------|
| 当日下跌 | 94% |
| 跌破 MA60 | 87% |
| KDJ 超卖 (K<30) | 60% |
| RSI<35 | 59% |
| 缩量 (量比<1) | 66% |
| 60日回撤>20% | 100% |

## 评分模型

满分 30 分：KDJ(5) + RSI(4) + MA60偏离(5) + 连跌天数(4) + 价格分位数(4) + 60日回撤(3) + MACD(2) + 成交量(2) + 布林(1)
